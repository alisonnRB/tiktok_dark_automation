# 🌑 TikTok Dark Automation

Automatize tarefas no TikTok com scripts personalizados! Este projeto visa facilitar interações automatizadas com a plataforma, utilizando diversas ferramentas para operações eficientes e seguras.

---

## 📌 Funcionalidades

✅ Automção de interações no TikTok<br>
✅ Scripts para abertura de sites e digitação automática<br>
✅ Integração com ChatGPT para respostas automatizadas<br>
✅ Execução de algorítmos personalizados

---

## 🚀 Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/alisonnRB/tiktok_dark_automation.git
   ```

2. Acesse o diretório do projeto:
   ```bash
   cd tiktok_dark_automation
   ```

3. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
   *Caso o arquivo `requirements.txt` não esteja presente, instale manualmente as bibliotecas necessárias conforme o código.*

## ▶️ Como Usar

Execute o script principal:
